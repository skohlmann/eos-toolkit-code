SimilarityFactory

In a concordance system the length of a document may not be crucial. The main implementation of the Lucene length of the document is considered. The shorter document is the more important document. To suppress this attitude the default similarity implementation is NormedLengthSimilarity which sets the lengthNorm(String,int) always to 1.0f.

LuceneDocumentCreator

To decouple the creation of the Lucence documents use instances of LuceneDocumentCreator. The newInstance(Configuration) factory method creates a new instance.

AnalyzerFactory

Its also possible to replace the Lucene analyzer. User the AnalyzerFactory with the newInstance(Configuration) factory method to create a new instance.
